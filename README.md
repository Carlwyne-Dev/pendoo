# Pendoo 🌱

**The peaceful pending list.**

Pendoo is a beautifully designed, serene task manager built to help you track things you are waiting on from others without the anxiety of traditional to-do lists.

## Features
- **Peaceful Design:** An intuitive, anxiety-free interface featuring a minimalist design and soft earthy colors.
- **Smart Deadlines:** Set expected dates or follow-up reminders. Pendoo gently notifies you when it's time to act.
- **Clarity History:** Move resolved items to your history and watch your 'Past clarity' grow over time.
- **Beautiful Soft Prompts:** Thoughtful UX flows for requesting system permissions gracefully.
